# 诗笺 Chinese Poetry Vue

一款基于 Vue 3、Vite 与[诗泉 API](https://poetry.palemoky.com/)开发的中国古典诗词网页。

## 功能

- 随机诗词推荐
- 按朝代和体裁筛选
- 全文、标题、正文及作者搜索
- 简繁体切换
- 本地收藏与一键复制
- 响应式移动端设计

## 语言文件

- 简体中文是默认语言，文案直接维护在 `src/App.vue` 的 `zhHans` 对象中。
- 繁体中文独立维护在 `src/locales/zh-Hant.js`。
- 繁体语言包会与默认简体文案合并；新增字段若暂未翻译，会自动回退到简体，避免界面出现空白。
- 诗词正文的简繁内容仍由诗泉 API 的 `lang=zh-Hans / zh-Hant` 参数提供，不由前端语言文件转换。

## API 代理

生产环境通过 `functions/api/[[path]].js` 将同源 `/api/*` 请求代理到诗泉 API，避免浏览器受到第三方接口 CORS 限制。Cloudflare Pages 部署时会自动识别该 Functions 目录，无需额外环境变量。

### 超时与重试

- 浏览器端 API 请求超时为 10 秒。
- Cloudflare Pages Function 请求上游诗泉的超时同样为 10 秒。
- 网络错误、HTTP 408、429 与 5xx 错误最多自动重试 2 次，并采用递增等待时间。
- 参数错误、权限错误、404 等确定性错误不会重试。

### API 状态页

访问 [`/status.html`](/status.html) 可检测本站 API 代理及诗泉上游服务状态。状态页会并发检测随机诗词、统计、朝代、体裁与诗词目录接口，并显示 HTTP 状态和响应延迟。
